🇫🇷 FRANÇAIS
-------------------------------------------------------------------------------

Mini Dictaphone V1 est un dictaphone vocal flottant, toujours au premier plan,
avec reconnaissance vocale 100 % hors-ligne via Whisper. Aucune donnée (audio
ou texte) ne quitte votre ordinateur.

Pensé comme un utilitaire discret (style PureRef), il flotte au-dessus de vos
autres applications, prêt à capter votre voix à tout moment. Interface
disponible en 6 langues (français, anglais, espagnol, portugais, allemand,
italien). Raccourci épinglable à la barre des tâches Windows 11.

Fonctionnalités principales :
  • Dictée vocale hors-ligne (Whisper, GPU)
  • Fenêtre flottante Always-On-Top
  • Sauvegarde automatique par jour + sauvegarde audio
  • Gestionnaire de modèles (1 clic)
  • Correcteur d'orthographe 100 % local (LanguageTool, 3 modes + dico perso)
  • Traduction vers l'anglais (option native Whisper)
  • Sortie webhook n8n générique (envoi auto vers n8n/LLM/Discord/Notion…)
  • Filtrage automatique des bruits parasites
  • Insertion de fichiers (texte, Excel, Word, PDF)
  • 100 % privé, aucune télémétrie
  • 100 % portable (clé USB) — tout dans un seul dossier

Technologies : Electron, Whisper (Const-me), JavaScript natif.


🇬🇧 ENGLISH
-------------------------------------------------------------------------------

Mini Dictaphone V1 is a floating voice dictaphone, always on top, with 100%
offline speech recognition via Whisper. No data (audio or text) ever leaves
your computer.

Designed as a discrete utility (PureRef style), it floats above your other
applications, ready to capture your voice at any time. Interface available in
6 languages (French, English, Spanish, Portuguese, German, Italian). Shortcut
pinnable to Windows 11 taskbar.

Main features:
  • Offline voice dictation (Whisper, GPU)
  • Always-On-Top floating window
  • Automatic save by day + audio save
  • Model manager (1 click)
  • 100% local spellchecker (LanguageTool, 3 modes + personal dictionary)
  • English translation (native Whisper option)
  • Generic n8n webhook output (auto-send to n8n/LLM/Discord/Notion…)
  • Automatic background noise filtering
  • File insertion (text, Excel, Word, PDF)
  • 100% private, no telemetry
  • 100% portable (USB stick) — all in a single folder

Technologies: Electron, Whisper (Const-me), native JavaScript.
