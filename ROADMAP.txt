A. CORRECTEUR D'ORTHOGRAPHE (PRIORITÉ HAUTE)
-------------------------------------------------------------------------------
  [ ] Intégrer LanguageTool en LOCAL (serveur sur le PC)
  [ ] Téléchargement automatique de Java portable (Temurin, licence libre)
  [ ] Téléchargement automatique de LanguageTool (~150 Mo)
  [ ] Lancement du serveur en arrière-plan (localhost:8081)
  [ ] Bouton "Installer le correcteur" dans le panneau ⚙️
  [ ] Correction automatique après chaque transcription
  [ ] Décision à prendre : fallback si LanguageTool est off ?

  NOTE : Le dictionnaire Windows/Hunspell a été écarté (trop mauvais en
  français pour les accords). LanguageTool local est le choix retenu.
  Alternative évoquée : LM Studio (déjà présent pour Jarvis) mais ne
  marche que si Jarvis tourne.


A2. TRADUCTION VERS L'ANGLAIS (PRIORITÉ MOYENNE)
-------------------------------------------------------------------------------
  [ ] Bouton "🌐 Traduire" pour convertir la dictée en anglais
  [ ] Utilise l'option native --translate de Whisper (zéro dépendance)
  [ ] Le texte transcrit est affiché en anglais au lieu de la langue source
  [ ] Idéal pour : remplir des formulaires/prompt sur des sites anglais
      (modèles IA, services), ou pour des utilisateurs ne parlant pas
      anglais mais voulant communiquer dans cette langue
  [ ] Bascule : on dicte en français → texte final en anglais

  NOTE : Whisper ne sait traduire QUE vers l'anglais (pas vers d'autres
  langues). Pour une traduction multilingue (FR↔ES, etc.), il faudrait une
  librairie locale supplémentaire (Argos Translate, ~500 Mo) — écarté pour
  l'instant, à revoir si besoin.

##############################################################################
#                         DÉCISIONS ARCHITECTURALES                         #
##############################################################################

- 100% OFFLINE pour la transcription (audio + texte ne quittent jamais le PC)
- Zéro dépendance externe pour le cœur (Electron seul au départ)
- Librairies ajoutées : xlsx, mammoth, pdf-parse (toutes 100% locales)
- Mode "enregistrer puis transcrire" (pas temps réel) pour la robustesse
- Dossier Whisper/ isolé pour ne pas interférer avec d'autres projets
- Open source : traductions et code perfectibles, contributions bienvenues
- Formats audio en WAV (qualité sans perte, simple)


##############################################################################
#                           ÉTAT DU PROJET - V1                             #
###############################################################################

Le Mini Dictaphone V1 est FONCTIONNEL pour son usage principal :
dictée vocale 100% locale en français (et 5 autres langues), avec
sauvegarde, historique, audio, insertion de fichiers.

Les fonctionnalités manquantes (correcteur, Jarvis) sont des évolutions
qui n'empêchent pas l'usage quotidien.
